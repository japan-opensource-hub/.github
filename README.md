# Japan Open Source Hub

**オープンソースに参加し、育てる社会へ**

IPAは日本社会全体のオープンソースへの参加力・OSS活用力を底上げするための中核的な推進主体としての役割を果たします。日本社会が技術的自立を確立し、ソフトウェアエンジニアリングの変革（ソフトウェアモダナイゼーション）を推進するための技術基盤として、OSSを創造し、育て、支える文化の醸成・制度の設計・人材の育成などに寄与することが、私たちの使命です。

OSSに関わる主体は開発者だけでなく、企業・行政機関・教育機関と多岐に渡るプレーヤー自身であり、IPAはその自律的な行動を後押しする伴走者の立場で機能します。

🔗 IPA公式ウェブサイトにて情報発信中 [Japan Open Source Hub](https://www.ipa.go.jp/digital/kaihatsu/oss/index.html)

## ポリシー

本Organizationは、IPAのオープンソース推進活動の一環としてIPAデジタル基盤センターが設立したものであり、以下を基本原則とします。

- 成果物「OSPOスターターキット」および関連リソースを公開し、産業界・公共分野を含む社会に還元する。
- 発信する情報は、一次ソース（Markdown形式のテキスト及び関連コード）を基盤とし、PDFやHTML形式など複数フォーマットでの利活用を可能にする。
- 発信対象は、IPAオープンソース推進から発信する中で継続的なアップデートが必要であり、かつ産業界や公共分野に広くニーズが見込まれるドキュメントとする。
- 外部からの改善提案（Issue, Pull Request）を建設的に受け入れ、オープンソースコミュニティの発展に寄与する。

## 運用方針

### リポジトリ公開方針

- ワークショップ期間中：プライベートリポジトリとして運用する。
- ワークショップ終了後：レビューを経て公開する。

### アクセス管理

- 管理者：IPA担当者
- プライベートリポジトリのコントリビュータ：ワークショップ参加者（限定期間）

### 公開後の運用

- IssueやPull Request を広く受け入れる。
- 改善提案はオープンな議論を通じて検討する。
